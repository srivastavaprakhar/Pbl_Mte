Stock Market Prediction from Candlestick Images

"Can deep learning models predict stock market movements by analyzing visual patterns in candlestick chart images? This project transforms 6.15 million sequential OHLCV windows into synthetic images and trains a CNN to detect price movement signals."

Problem Statement

The Problem: Retail and institutional investors struggle to predict short-term price movements in Indian stock markets, particularly for Nifty 100 stocks with high-frequency 15-minute interval data. Traditional numerical models (LSTM, ARIMA) fail to capture the complex spatial and visual patterns that experienced traders identify in candlestick charts‚Äîpatterns like doji, hammer, engulfing, and support/resistance levels. Furthermore, training on isolated snapshots causes models to see disconnected patterns without understanding transitions between market states.

The Solution: This project implements a sliding window protocol (size=5) to provide temporal context, generating 6.15 million overlapping samples from 7+ years of OHLCV data (Jan 2015 - Feb 2022). Each sample captures five consecutive candlesticks, enabling the model to learn short-term dynamics and contextual relationships. These windows are converted into 150√ó150 images with integrated volume indicators, and a CNN classifies binary price movements at scale and speed impossible for manual analysis.

Key Challenges

  • ‚Ä¢ High volatility in 15-min interval data
  • ‚Ä¢ Isolated snapshots lack temporal context
  • ‚Ä¢ Preventing look-ahead bias in time-series
  • ‚Ä¢ Scale normalization for comparable inputs

Literature Review

Existing work falls into three lines: (1) CNN-based approaches convert OHLC to candlestick images using ResNet/VGG for daily predictions on non-Indian markets, achieving 90‚Äì99% accuracy without explicit pattern detection. (2) GAF-based methods encode sequences into mathematical images with CNN/CNN-LSTM for classic pattern recognition at hourly/daily resolutions. (3) Forex TA-Lib integration uses 61 candlestick patterns with 15-min sliding windows and CNNs for trend prediction. For Indian markets, research focuses on daily NIFTY50 data evaluating pattern profitability but lacks combining TA-Lib filtering, intraday 15-min windows, and standardized candlestick images with CNNs.

Research Gap / Innovation

Unlike existing approaches, SMPS targets the Indian market (Nifty 100) with intraday 15-minute data, uses sliding window protocol (6.15M samples) to preserve temporal context, and applies Z-score normalization for stable training. The modular pipeline enables both batch and streaming training for real-time trading applications, addressing gaps in Indian market-specific high-frequency candlestick pattern analysis.

System Methodology

Dataset / Input

Nifty 100 stocks from Indian Stock Market (Jan 2015 - Feb 2022). Data sampled at 15-minute intervals with OHLCV features. Sliding window protocol (size=5) generates 6.15M samples, each containing 5 consecutive candles to preserve temporal context. Shape: (6,150,190, 5, 5)

Model / Architecture

PyTorch-based CNN (SimpleCNN) trained on 150√ó150 RGB images. Each image encodes 5-candle window with price action (120px) and volume (30px). Binary classification for up/down movements. Z-score normalized inputs (mean‚âà0, std‚âà1) for stable training.

Sliding Window Protocol

Why Sliding Windows?
Without sliding windows, the model sees disconnected patterns and cannot learn transitions between market states. Each isolated snapshot lacks temporal context, leading to vague training.

Window Size = 5
Captures short-term price dynamics and local contextual relationships between consecutive candlesticks. Allows effective comparison between recent historical movements and current price action while maintaining high sample count.

Generated Samples: 6,150,190
Each Sample: 5 consecutive candles
Label: Next candle's price direction

Performance Analysis

The model achieved 53.4% test accuracy on binary price movement classification (up/down). While this is marginally above random baseline (50%), it demonstrates that the CNN successfully learned some visual patterns from candlestick images. The quick convergence and early stopping at epoch 4 indicates the model found stable patterns but reached capacity limits of the architecture. Future improvements could include: deeper networks, attention mechanisms, ensemble methods, or incorporating additional technical indicators.
